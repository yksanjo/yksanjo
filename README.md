## What I Build

I build open-source infrastructure that makes AI agents more useful, observable,
and safe: coding assistants, MCP discovery and memory, multi-agent operations,
and transaction-capable Solana tools.

My strongest work is designed as software you can install and run—not just a
concept demo. The flagship projects below have automated tests, documented
security boundaries, and reproducible local setup.
